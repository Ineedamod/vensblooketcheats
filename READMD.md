Professional Blooket Dashboard with integrated GitHub cheats and local bot deployment.

## Setup
1. Download this repo.
2. Run `npm install` in your terminal.
3. Run `node server.js`.
4. Open `http://localhost:3000` in your browser.

## Features
- **Panic Button**: Instant redirect to Google Backpack.
- **Auto-Injection**: Fetches the latest Glixzzy scripts from GitHub.
- **Bot Control**: Deploy and kill local Node.js bot processes.
